SSG - Simple Sass Grid

Including

You can either include the grid by using the grid.scss, example:

@import "grid.scss";

... or by importing all partials by them self (this is what grid.scss does), example:

@import "grid_partials/grid-variables.scss"; @import "grid_partials/grid-mixins.scss"; @import "grid_partials/grid-engine.scss";

Variables

U can setup the variables anyway you like. This is useful to change support for other systems like bootstrap. Or if you want to add more breakpoints.

Example of grid-variables.scss:

$grid-prefix: ''; // Example: set 'grid-' to make wrapper, row and columns grid-* $grid-col-name: 'col'; $grid-row-name: 'row'; $grid-wrapper-name: 'wrapper'; $grid-columns: 12; $grid-max-width: 65em; $grid-col-padding: 1em; $grid-breakpoints: ( ('xs', 0px), ('sm', 768px), ('md', 992px), ('lg', 1200px) );

Using the mixins

Here you can see how to use the mixins

Media mixins

There are three media mixins

  • media-min($breakpoint_name)
  • media-max($breakpoint_name)
  • media-is($breakpoint_name)

Here is an example how to use them

Code

.mixin-example-is { background-color: #00EEEE; @include media-is('sm') { background-color: #EEEE00; } @include media-max('xs') { background-color: #EE00EE; } }
